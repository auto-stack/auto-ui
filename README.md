## AutoUI

AutoUI是AutoStack的UI库，基于Auto和Rust语言实现。

AutoUI的底层调用的是[Zed编辑器](https://zed.dev/)的[GPUI图形库](https://www.gpui.rs/)。

AutoUI的设计和实现参考了[GPUI-Componets组件库](https://github.com/Zed-Editor/gpui-components)。
