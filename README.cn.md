## Auto UI

AutoUI是基于Auto语言和Rust语言实现的UI框架。

AutoUI底层采用[GPUI](https://www.gpui.rs/)图形库，
并参考[GPUI-Component](https://github.com/longbridgeapp/gpui-component)进行组件设计和实现。
